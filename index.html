Git commands

  1. git --version
  2. git init : creates a repository
  3. git config --global user.name /* Your name here */
  4. git config --global user.email /*Your email here*/
  5. git add .
  6. git commit -m "message"
  7. git remote add origin https://github.com/swarha22/git-example.git
  8. git branch -M main
  9. git push -u origin main
